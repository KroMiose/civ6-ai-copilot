# 玩家使用指南

日常安装和第一次使用请优先看仓库根目录的 `README.md`。本文只补充普通玩家最容易遇到的使用流程。

## 基本流程

1. 在 Steam Workshop 订阅 `Civ6 AI Copilot`。
2. 在 Civilization VI 的 `额外内容` 中启用 Mod。
3. 进入对局后，点击左上角副官入口。
4. 在「战情简报」里点击「汇总本回合」。
5. 回到 Codex、Claude Code 或其他本地 Agent，请它使用 `civ6-ai-copilot` skill 读取最新战情。

## 推荐提问

```text
我刚刚在 Civ6 的战情简报中汇总了本回合情报。请使用 civ6-ai-copilot skill 读取最新战情，告诉我本回合应当如何行动
```

也可以直接问：

```text
这些城市接下来分别造什么？
```

```text
下一条科技和市政怎么走？
```

```text
我现在适合开战吗？如果不适合，前线怎么防？
```

```text
政策卡怎么换，资源该怎么用？
```

## 信息不足时

如果 Agent 说缺少情报，回到「战情简报」点击它指定的按钮，再继续原来的对话即可。

- 「更新地图情报」：地图、前线、海军、侦察、定居和战争判断。
- 「城市运营」：逐城生产、住房、区域和产出。
- 「军事态势」：单位行动、防守和调兵。
- 「科技市政」：科技、市政、尤里卡和鼓舞路线。
- 「政体政策」：政体、政策槽和已插政策。
- 「资源库存」：战略资源、奢侈品、升级、维护和交易。
- 「公开外交」：已遇见文明、公开关系和公开军事分。
- 「完整战情简报」：首次使用、版本变化、诊断异常或多个专题都需要更新。

## 多人局

多人房间如果限制 UI 或 utility Mod，请先遵守房间规则。这个项目的目标是做回合规划辅助，不是作弊工具。
