# Civ6 AI Copilot v0.1.0

首个公开测试版。

## 包含内容

- Civilization VI 被动 UI/utility Mod：提供游戏内「战情简报」面板。
- Agent Skill：读取战情材料，辅助城市、科技、市政、政策、军事、海军、资源和定居判断。
- 本地工具：bridge、tuner-bridge、doctor、preflight、summary、handoff、map render、release gate。
- Steam Workshop 安装入口和 GitHub 手动安装包。

## 安装

1. 在 Steam Workshop 订阅 `Civ6 AI Copilot`。
2. 启动 Civilization VI，在 `额外内容` 中启用 Mod。
3. 把下面的请求发给 Codex 或 Claude Code 等本地 Agent：

```text
请从 https://github.com/KroMiose/civ6-ai-copilot 安装或更新 civ6-ai-copilot Agent Skill 和本地助手工具。
优先使用最新 GitHub Release；如果还没有 Release，就从仓库安装 skill/ 和必要的 tooling。
完成后告诉我 skill 版本、本地工具目录，以及是否需要重启客户端或开启新对话。
```

## 验证状态

- 自动 gate：Mod、Skill、schema、fixture、fake Lua.log bridge、doctor、preflight、summary、map render、handoff、privacy check。
- 手工 gate：Windows Civ6 实机加载、双人多人公平检查、Mac Agent handoff 仍需按 `tests/manual/` 模板记录最终证据。

## 注意

- 这是 pre-release。建议先用于测试局、单机局或已获得同局玩家同意的多人局。
- Mod 不修改规则、地图、单位、资源、外交、生产、存档或网络同步状态。
