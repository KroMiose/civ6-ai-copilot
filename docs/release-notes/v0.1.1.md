# Civ6 AI Copilot v0.1.1

本次是面向游戏内体验的补丁版。

## 更新内容

- 优化「战情简报」面板布局，让状态、进度和操作按钮更整齐。
- 将地图扫描、校验和快照写入拆分到多帧执行，缓解汇总战情时游戏界面和声音长时间卡住的问题。
- 自动汇总回合开始时改为排队执行，并显示当前汇总阶段与进度。
- 改进 Mod 打包校验，确保发布包使用新的 UI 文件结构。

## 安装

1. 在 Steam Workshop 订阅或等待 `Civ6 AI Copilot` 自动更新。
2. 启动 Civilization VI，在 `额外内容` 中确认 Mod 已启用。
3. 把下面的请求发给 Codex 或 Claude Code 等本地 Agent：

```text
请从 https://github.com/KroMiose/civ6-ai-copilot 安装或更新 civ6-ai-copilot Agent Skill 和本地助手工具。
优先使用最新 GitHub Release；如果还没有 Release，就从仓库安装 skill/ 和必要的 tooling。
完成后告诉我 skill 版本、本地工具目录，以及是否需要重启客户端或开启新对话。
```

## 验证状态

- 自动 gate：Mod、Skill、schema、fixture、fake Lua.log bridge、doctor、preflight、summary、map render、handoff、privacy check。
- 手工 gate：建议在真实 Civ6 对局中确认左上副官入口、手动「汇总本回合」和「回合开始自动汇总」均能正常显示进度并完成导出。

## 注意

- 这是 pre-release。建议先用于测试局、单机局或已获得同局玩家同意的多人局。
- Mod 仍是被动 UI / Utility Mod，不修改规则、地图、单位、资源、外交、生产、存档或网络同步状态。
