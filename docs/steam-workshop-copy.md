# Steam Workshop Copy

Steam Workshop 标题和描述按语言保存。发布时分别选择 English 和 Simplified Chinese，不要把中文正文写进 English 语言页。

## English

Title:

```text
Civ6 AI Copilot
```

Description:

```text
[h1]Civ6 AI Copilot[/h1]
[b]Turn your Civilization VI position into an AI-ready briefing.[/b]

[h2]What it helps with[/h2]
[list]
[*]Get clearer turn-priority suggestions.
[*]Review city production, tech and civics, policy cards, resources, and military defense.
[*]Think through exploration, naval movement, settling spots, and war risk.
[*]Spend less time re-describing the board state, screenshots, and turn context.
[/list]

[h2]Install the Mod[/h2]
[olist]
[*]Subscribe on Steam Workshop.
[*]Start Civilization VI.
[*]Enable Civ6 AI Copilot in Additional Content.
[*]Start or load a game.
[/olist]

[h2]Install the AI Skill[/h2]
Steam Workshop only installs the Civ6 Mod. To let your AI agent read the briefing, send this to Codex, Claude Code, or another local agent:

[quote]Please install or update the civ6-ai-copilot Agent Skill and local helper tools from https://github.com/KroMiose/civ6-ai-copilot.
Prefer the latest GitHub Release. If no Release is available yet, install the skill/ folder and required tooling from the repository.
When finished, tell me the skill version, local tooling directory, and whether I need to restart the client or open a new chat.[/quote]

[h2]First Use[/h2]
[olist]
[*]Enter a Civ6 game and click the Copilot entry in the upper-left UI.
[*]In the briefing panel, choose the full turn briefing action.
[*]Return to your AI agent and ask it to use the civ6-ai-copilot skill to read the latest briefing.
[/olist]

[h2]Safety[/h2]
This is a passive UI / Utility Mod. It does not change rules, maps, units, resources, diplomacy, production, saves, or network sync state. For multiplayer games, follow your room rules and player agreement.
```

Changenote:
